Trait that Actors instance have to implement for a Swarm so their internally stored instance RawID can be gotten and set.

Furthermore, an Actor has to implement Compact, so a Swarm can compactly store each Actor's potentially dynamically-sized state.

This trait can is auto-derived when using the kay_codegen build script.
